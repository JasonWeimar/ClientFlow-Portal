## Design Decisions

- **Color tokens**: All colors defined as named Figma Color Styles,
  mapped 1:1 to tailwind.config.ts tokens
- **Typography**: Inter for UI, JetBrains Mono for code elements
- **Spacing**: 8px grid system throughout
- **Components**: 28 Figma components extracted and mapped to React files
